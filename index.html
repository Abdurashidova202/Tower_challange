EASY CLASS ACTIVITY

Tower Challenge (60 Minutes)

Work in a team, Build the tallest tower that can hold a small object for 10 seconds.

Class Timer

60:00

Now: 0-5 min: Teacher explains the challange


What You Need (per team)

  • 20 straws or 10 paper sheets
  • Tape
  • String
  • Scissors
  • One small object (coin/eraser)

Simple Rules

  1. Your tower must stand on its own
  2. It must hold the object at the top
  3. It must stay up for 10 seconds
  4. Use only the materials given.

Today Plan

0-5 min: Teacher explains the challenge
5-15 min: Team plan and quick sketch
15-35 min: Build your first tower
35-45 min: Test and improve
45-55 min: Final build time
55-60 min: Final test and class winners
